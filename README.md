# Personal Neovim Plugin Archive

This repository contains archived versions of Neovim plugins used in my personal configuration.
Each plugin is stored at a specific commit to ensure reproducibility and stability.

## Structure

```
plugin-name/
└── commit-hash/
    └── [plugin files]
```

## Usage

This is a personal archive. To use these plugins with lazy.nvim:

```lua
{
    dir = "~/archive/push/plugin-name/commit-hash",
    lazy = true,
}
```
